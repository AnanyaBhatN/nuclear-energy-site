# Nuclear Energy – Simplified ⚛️

Nuclear Energy – Simplified is an educational web application designed to explain nuclear energy concepts in a clear, structured, and visually engaging manner.  
The project focuses on making complex scientific ideas accessible to students and beginners through a modern user interface and well-organized content.
---

## 🌐 Live Demo
ananyabhatn.github.io/nuclear-energy-site/

---

## 📌 Project Overview
This website covers essential topics related to nuclear energy, including:
- Basic atomic structure and nuclear physics
- Nuclear fission and fusion
- How nuclear reactors work
- Benefits and challenges of nuclear energy
- Common myths and misconceptions
- Present and future nuclear technologies
- Curated resources for further learning

The application is implemented as a single-page website with smooth navigation and responsive design.

---

## ✨ Key Features
- Single-page application with JavaScript-based navigation
- Clean and modern UI using Tailwind CSS
- Dark mode support
- Responsive design for mobile and desktop devices
- Well-structured educational content
- External resources for extended learning
